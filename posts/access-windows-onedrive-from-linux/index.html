在 Linux 存取 Windows 分割區的 OneDrive 資料夾

2023/02/13

Linux

朋友存了不少檔案在 OneDrive 上，問怎麼在 Linux 直接存取 Windows 上的 OneDrive 資料夾（會顯示 unsupported reparse point 錯誤訊息），這篇文就誕生了。

  1. （若未安裝）安裝 ntfs-3g（支援 FUSE 者後續掛載時不須 sudo）
  2. 從這裡下載插件
  3. 解壓縮後把相應平台的插件（如 linux-64 裡的 ntfs-plugin-9000001a.so）放到 /usr/lib/ntfs-3g/ 底下（可能需要自行建立資料夾）
  4. 使用 sudo ntfs-3g /分割區 /掛載點 或 GUI 掛載 Windows 分割區，例如：sudo ntfs-3g /dev/sda1 /mnt/windows。若已掛載則須先卸除 OneDrive 所在的 Windows 分割區

結果：可以讀寫所有 OneDrive 在 Windows 上的檔案，但不會同步。
