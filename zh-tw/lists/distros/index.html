Distros to begin

2022/10/10

There are a lot of Linux distributions to choose from. Here are some distros I like and recommand to start with:

Debian/Ubuntu Based

  • KDE neon
  • Mint
  • Zorin OS

Arch Based

  • Manjaro
  • CachyOS

Why not Ubuntu?

Since Ubuntu 22.04, it directs apt install to Snap by default, which is forced by Canonical and might lead to broken or unexpected packages. The default desktop layout (vertical teskbar and Activities) can also be somewhat confusing to newcomers.
