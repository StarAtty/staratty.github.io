遊戲

Pixel TD

像素化塔防遊戲，大二下（2023 年）在一個月裡用 Makecode Arcade 做的。波數無限，敵人會變強。鍵盤的 Z 跟 X 是模擬器的 A 跟 B。硬體和瀏覽器會影響塔的性能。

北藝大 Geoguessr

看題目的圖片猜在學校的哪裡。畢業前一週做的。

音熊

2D 平台解謎遊戲，大二下跟幾個同學和動畫系學姐做的。故事的前情提要是音樂森林裡的村落舉辦一年一度的音樂祭時，突然來了可怕的龍捲風，主角醒來後發現周遭一片狼藉……。

學期初我自己從零開始學 Unity，同學做音樂、聲音、故事等，動畫系學姐畫圖。期末前一個月左右在北流報告演示給好幾個老師跟（當時的）全系看，順利收集完音符、啟動機關時幾十個人一片歡聲雷動。但是，要學期末時我的系統壞掉，重灌完才發現專案檔因為雲端備份滿了沒備到而丟失了不少進度，最後好像在兩三天內修回一個月內邊學邊做的東西，但基於遊戲沒有要繼續做下去，所以期末報告後就沒在維護了。
